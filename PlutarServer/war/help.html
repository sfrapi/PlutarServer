Plutar - Aide

Plutar est une application de démonstration permettant de programmer le dépôt de messages pré-enregistrés, simultanément sur plusieurs boites vocales de clients SFR
Attention : Les destinataires des messages doivent avoir donnés leur consentement pour recevoir des messages avant que celui-ci ne soit déposé sur leur boite.


Le message peut être créé
- grâce au micro du mobile
- en synthèse vocale depuis un texte saisi
- depuis un fichier disponible sur le téléphone ou sur le web
